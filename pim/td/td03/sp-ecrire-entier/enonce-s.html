Écrire un entier

Énoncé

Notre objectif est de définir un sous-programme qui permet d’écrire un entier (dans le terminal) en n’utilisant que la procédure qui permet d’écrire un seul caractère, Put(Character).

On se limitera dans un premier temps aux entiers positifs en écrivant deux versions du sous-programme, la première récursive (Ecrire_Recursif), la seconde itérative (Ecrire_Iteratif).

Ensuite, on écrira un sous-programme Ecrire qui traite les entiers relatifs, positifs ou négatifs donc. Bien sûr, on réutilisera les sous-programmes précédents, par exemple Ecrire_Recursif, pour ne pas dupliquer du code.

Déroulement :

  1. Coder la version récursive et la tester (seuls les derniers tests, à partir des nombres négatifs, devraient échouer).
  2. Coder la version itérative et la tester (les mêmes tests devraient réussir).
  3. Coder le sous-programme Ecrire (tous les tests devraient réussir).
  • Remarque : Réussir les 3 derniers tests est optionnel. Ils permettent de voir si les risques de dépassement de capacité des entiers sont bien gérés.

Contrainte :

  • On n’utilisera que l’écriture d’un caractère, Put(Character), puisque le but est de réécrire l’écriture d’un entier.

Indications :

  1. Les caractères étant consécutifs, on peut obtenir le caractère qui correspond à un chiffre en faisant :
    • En algorithmique : Chr (Ord ('0') + Chiffre)
    • En Ada : Character'Val (Character'Pos('0') + Chiffre)
  2. Dans la version récursive, on notera qu’il est facile d’obtenir le chiffre des unités d’un nombre et ce nombre privé de ces unités. Par exemple pour 421, on obtient facilement 1 et 42.
  3. Il faut utiliser la méthode des raffinages !

Entrée

un entier, l’entier correspondant aux caractères saisis.

Sortie

les caractères qui constituent un entier

Exemples

entier      ->  caractères
----------------------------
421         ->  421           -- plusieurs chiffres constituent le nombre
1           ->  1             -- un seul chiffre
0           ->  0             --    "      "
12300       ->  12300         -- avec des 0 à la fin
10001       ->  10001         -- avec des 0 au milieu
123456789   ->  123456789     -- beaucoup de chiffres

-5          ->  -5            -- nombre négatif avec un seul chiffre
-235        ->  -235          -- nombre négatif avec plusieurs chiffres
-123456789  ->  -123456789    -- nombre négatif avec beaucoup de chiffres

1234567890  ->  1234567890    -- nombre maximal de chiffres pour les Integer d'Ada
2147483647  ->  2147483647    -- le plus grand des entiers (Integer'LAST)
-2147483648 ->  -2147483648   -- le plus petit entier Integer'FIRST

with Ada.Integer_Text_IO;  use Ada.Integer_Text_IO;

    -- Nécessite : Nombre >= 0  -- Nombre positif

    end Ecrire_Recursif;


    -- Écrire un entier.
    -- Paramètres :
    --    Nombre : l'entier à écrire
    -- Nécessite : Nombre >= 0  -- Nombre positif
    -- Assure : -- Nombre est écrit
    procedure Ecrire_Iteratif (Nombre: in Integer) is

        Ecrire_Recursif (Nombre);
    end Ecrire_Iteratif;


    -- Écrire un entier.
    -- Paramètres :
    --    Nombre : l'entier à écrire
    -- Nécessite : ---
    -- Assure : -- Nombre est écrit
    procedure Ecrire (Nombre: in Integer) is
    begin
        -- TODO : remplacer ce code quand Ecrire_Iteratif sera terminé.
        Ecrire_Recursif (Nombre);
    end Ecrire;


    Un_Entier: Integer;     -- un entier lu au clavier
    Message: constant String := "L'entier lu est ";
begin
    -- Demander un entier
    Put ("Un entier : ");
    Get (Un_Entier);

    -- Afficher l'entier lu avec les différents sous-programmes
    if Un_Entier >= 0 then
        -- L'afficher avec Ecrire_Recursif
        Put (Message & "(Ecrire_Recursif) : ");
        Ecrire_Recursif (Un_Entier);
        New_Line;

        -- L'afficher avec Ecrire_Iteratif
        Put (Message & "(Ecrire_Iteratif) : ");
        Ecrire_Iteratif (Un_Entier);
        New_Line;
    else
        Put_Line ("Le nombre est négatif. "
            & "On ne peut utiliser ni Ecrire_Recursif ni Ecrire_Iteratif.");
    end if;
    --  L'afficher avec Ecrire
    Put (Message & "(Ecrire) : ");
    Ecrire (Un_Entier);
    New_Line;
end Ecrire_Entier;
