Majeure ?

Énoncé

Déterminer si une personne est majeure.

Contrainte :

  • On n’utilisera pas de décision (seulement des expressions booléennes).
  • On définira une constante Age_Majorite initialisée à 18.

Entrée

  • Age : un entier, l’âge de la personne. L’âge doit être positif.

Sortie

  • Est_Majeure : un booléen qui dit si la personne est majeure ou non

Exemples

Age  ->  Est_Majeure
-----------------
 0   ->  Non
25   ->  Oui
77   ->  Oui
17   ->  Non
18   ->  Oui

Squelette

-- Demander l'age de Jules
with Ada.Text_IO;          use Ada.Text_IO;
with Ada.Integer_Text_IO;  use Ada.Integer_Text_IO;

-- Afficher si une personne est majeure
procedure Majorite is

    Age: Integer;           -- Age de la personne
    Est_Majeure: Boolean;   -- Vrai si la personne est majeure
begin
    -- Demander l'age de la personne
    Get (Age);

    -- Déterminer la majorité de cette personne
    ...

    -- Afficher la somme des versements
    if Est_Majeure then
        Put ("Oui");
    else
        Put ("Non");
    end if;
    New_Line;
end Majorite;
