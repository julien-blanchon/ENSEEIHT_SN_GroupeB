Tarif d’une place

Énoncé

Écrire un programme qui calcule le tarif d’une place.

Le tarif normal de la place est de 12,60 €. Les enfants (moins de 14 ans) paient 7 €. Les séniors (65 ans et plus) paient 10,30 €.

Entrée

age : un entier

Sortie

tarif : un réel

Exemples

age  ->  tarif
-----------------
25   ->  12.60
 8   ->  7.0
75   ->  10.30
14   ->  12.60

Squelette

with Ada.Text_IO;          use Ada.Text_IO;
with Ada.Integer_Text_IO;  use Ada.Integer_Text_IO;
with Ada.Float_Text_IO;    use Ada.Float_Text_IO;

-- Afficher le tarif d'une place
procedure Tarif_Place is

    Age: Integer;  -- l'âge de la personne
    Tarif: Float;  -- le tarif à appliquer
begin
    -- Demander l'age
    Get (Age);

    -- Déterminer le tarif de la place
    ...

    -- Afficher le tarif
    Put ("Le tarif pour ");
    Put (Age, 1);
    Put (" ans est ");
    Put (Tarif, Exp => 0, Aft => 2, Fore => 1);
    Put (" euros.");
    New_Line;

end Tarif_Place;
