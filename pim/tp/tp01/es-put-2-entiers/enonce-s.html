Écrire des entiers

Énoncé

Dans cet exercice, on veut écrire de différentes manières deux entiers lus au clavier. Les consignes sont directement données dans le programme où il faut remplacer « TODO; » par le instructions qui font ce qui est demandé dans le commentaire qui précède.

Conseil : Remplacer les « TODO; » les uns après les autres et valider votre programme à chaque remplacement. Si vous l’avez correctement traité, l’erreur sera sur le TODO suivant qu’il faudra alors remplacer. Dans le cas contraire, c’est que vous n’avez pas fait ce qui est attendu : il faut corriger avant de passer à la suite.

Rappel : Le module Ada.Integer_Text_IO fournit une procédure Put qui prend en première paramètre l’entier à écrire sur la sortie standard. Elle accepte un deuxième paramètre optionnel qui indique sur combien de position l’entier doit être affiché. Un troisième paramètre optionnel permet d’indiquer la base dans laquelle l’entier doit être affiché (base 10 par défaut).

Le module Ada.Text_IO fournit une procédure Put pour afficher une chaîne de caractères et une autre procédure Put pour afficher un caractère (surcharge). Une procédure New_Line permet de revenir à la ligne. Une procédure Put_Line se comporte comme Put mais en ajoutant un retour à la ligne.

Entrée

N1, N2 : deux entiers

Sortie

Les différents affichages demandés.

Exemples

Entrées :

N1 = 5 et N2 = -3

Affichage dans le terminal :

Un par ligne :
          5
         -3
Un par ligne, alignés contre la marge :
5
-3
La phrase demandée :
Le premier nombre est 5, le second -3.
Addition :
5 + -3 = 2

Squelette

with Ada.Text_IO;          use Ada.Text_IO;
with Ada.Integer_Text_IO;  use Ada.Integer_Text_IO;

procedure Signe_Entier is

    -- La procédure TODO affiche un message.
    procedure TODO is
    begin
        Put_Line ("Il faut remplacer TODO; par les instructions qui vont bien");
    end;


    N1, N2: Integer; -- Entier lu au clavier dont on veut connaître le signe

begin
    -- Demander les deux entiers N1 et N2
    Get (N1);
    Get (N2);

    -- Afficher N1 et N2 chacun sur une ligne
    Put_Line ("Un par ligne :");
    TODO;

    -- Afficher N1 et N2 alignés contre la marge
    Put_Line ("Un par ligne, alignés contre la marge :");
    TODO;

    -- Afficher « Le premier nombre est {N1}, le second {N2}."
    Put_Line ("La phrase demandée :");
    TODO;

    -- Afficher « {N1} + {N2} = {N1 + N2} »
    Put_Line ("Addition :");
    TODO;


end Signe_Entier;
