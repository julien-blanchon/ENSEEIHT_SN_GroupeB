Puissance entière d’un réel

Énoncé

Remarque : Si les formules mathématiques ne s’affichent pas bien, essayer avec firefox ou installer un greffon MathML pour votre navigateur.

Nous avons déjà écrit la puissance en version récursive et itérative. Le but de cet exercice est d’améliorer le calcul de la puissance de xnx^n en s’appuyant sur la constatation suivante.

  • Si nn est pair, nn s’écrit n=2pn=2p et on a xnx^n = (x2)p(x^2)^p
  • Si nn est impair, nn s’écrit n=2p+1n=2p + 1 et on a xnx^n = x(x2)px(x^2)^p

Par exemple, x9=x(x2)4=x((x2)2)2=x(((x2)2)2)1x^9 = x(x^2)^4 = x((x^2)^2)^2 = x(((x^2)^2)^2)^1 avec bien sûr x2=x*xx^2 = x*x puisqu’on ne peut pas utiliser l’opérateur puissance. Au total on aura fait 4 multiplications au lieu de 9.

On repartira du programme précédent et on proposera une nouvelle implantation de Puissance_Recursif et Puissance_Iteratif exploitant la propriété précédente. Tous les tests doivent réussir.

Entrée

  • un nombre réel
  • un exposant, entier

Sortie

  • nombre à la puissance exposant, un réel (version récursive et version itérative)

Exemples

nombre      exposant   ->  puissance
--------------------------------------------------
3.0         4          ->  81.0                  nombre positif, exposant positif
-2.0        5          ->  32.0                  nombre négatif, exposant positif impair
-2.0        6          ->  64.0                  nombre négatif, exposant positif pair
0.5         3          ->  0.125                 
4.0         -1         ->  0.25                  exposant négatif
17.0        1          ->  17.0                  exposant = 1
9.0         0          ->  1.0                   exposant = 0
0.0         0          ->  1.0                   exposant = 0
-1.0        100000     ->  1.0                   exposant assez grand, pair
-1.0        100001     ->  -1.0                       "       "        impair
0.0         -5         ->  Division par zéro
-1.0        1234567891 ->  1.0                   exposant très grand

Voici un exemple d’affichage

Un nombre réel : 3.00000
Un nombre entier : 4
Puissance (itérative) : 81.0000
Puissance (récursive) : 81.0000

Squelette

with Ada.Text_IO;          use Ada.Text_IO;
with Ada.Integer_Text_IO;  use Ada.Integer_Text_IO;
with Ada.Float_Text_IO;    use Ada.Float_Text_IO;

procedure Ecrire_Entier is

    function Puissance_Iteratif (Nombre: in Float ; Exposant : in Integer) return Float with
        Pre => Exposant >= 0 or Nombre /= 0.0;
        --! Ceci est une déclaration en avant.  La fonction Ecrire_Recursive
        --! est connue et peut être utilisée (sa signature suffit pour que le
        --! compilateur puisse vérifier l'appel)
        --!
        --! Remarque : les contrats ne doivent apparaître que lors de la première
        --! apparition.  Ceci évite les redondances toujours sources d'erreurs.


    -- Retourne Nombre à la puissance Exposant.
    function Puissance_Recursif (Nombre: in Float ; Exposant : in Integer) return Float with
        Pre => Exposant >= 0 or Nombre /= 0.0
    is
    begin
        return Puissance_Recursif(Nombre, Exposant);
    end Puissance_Recursif;


    function Puissance_Iteratif (Nombre: in Float ; Exposant : in Integer) return Float is
    begin
        return Puissance_Recursif (Nombre, Exposant);
    end Puissance_Iteratif;


    Un_Reel: Float;         -- un réel lu au clavier
    Un_Entier: Integer;     -- un entier lu au clavier
begin
    -- Demander le réel
    Put ("Un nombre réel : ");
    Get (Un_Reel);

    -- Demander l'entier
    Put ("Un nombre entier : ");
    Get (Un_Entier);

    -- Afficher la puissance en version itérative
    Put ("Puissance (itérative) : ");
    Put (Puissance_Iteratif (Un_Reel, Un_Entier), Fore => 0, Exp => 0,  Aft => 4);
    New_Line;

    -- Afficher la puissance en version récursive
    Put ("Puissance (récursive) : ");
    Put (Puissance_Recursif (Un_Reel, Un_Entier), Fore => 0, Exp => 0,  Aft => 4);
    New_Line;

end Ecrire_Entier;
