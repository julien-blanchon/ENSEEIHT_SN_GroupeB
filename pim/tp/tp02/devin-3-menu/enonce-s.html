Le jeu du devin : choisir le jeu

Énoncé

Dans cette troisième itération, on veut proposer à l’utilisateur de choisir entre les deux types de jeu : deviner le nombre ou le faire deviner. Ceci sera fait via un petit menu comme montré sur les exemples suivants. Attention, un retour à la ligne est fait avant l’affichage du menu et après le choix de l’utilisateur.

Bien sûr, on réutilisera les deux programmes (procédures) écrits précédement sans les modifier : il suffit de les copier/coller ici à la place des morceaux de code correspondants.

Conseil :

  • Bien lire et comprendre l’exemple ci-après.
  • Il faut impérativement respecter les messages affichés.
